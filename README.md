# 📊 Invest Workbench — ASURADA

미국시장 학습용 **실시간 대시보드 + 데모 트레이딩 + 매매일지** (토스 스타일 다크, 단일 HTML).

> ⚠️ 교육용 시뮬레이터입니다. 실제 돈이 오가지 않으며, 투자 권유가 아닙니다.

## 기능
| 섹션 | 내용 | 데이터 |
|---|---|---|
| ① 실시간 차트 | 심볼 검색 가능한 5분봉 차트 | TradingView (키 불필요) |
| ② 시장 개요 | 지수 / 환율·원자재 / 크립토 탭 | TradingView |
| ③ 지금 뜨는 종목 | 급등·급락·거래량 상위 | TradingView |
| ④ 섹터 히트맵 | S&P 500 "지금 뜨는 사업" 한눈에 | TradingView |
| ⑤ 주요 일정 | NFP 등 경제지표 캘린더 (investing.com 스타일) | TradingView |
| ⑥ 기관·내부자 동향 | SEC Form 4 내부자 매매 (미국판 "기관 동향" 프록시) | Finnhub |
| ⑦ 데모 트레이딩 | 가상 $10,000 · 매수/매도 · 평가손익 · 승률 | Finnhub |
| 매매일지 | **진입 근거 필수 입력** · 셋업 태그 · 복기 · CSV 내보내기 | localStorage |

## 셋업 (2분)
1. 이 저장소를 GitHub Pages로 배포 (Settings → Pages → main / root)
2. [finnhub.io/register](https://finnhub.io/register)에서 무료 API 키 발급
3. 페이지 하단 ⑧ 설정에 키 저장 (브라우저 localStorage에만 저장, 외부 전송 없음)

키 없이도 차트·캘린더 등 TradingView 위젯은 전부 동작. 키는 시세 조회·데모 매매·내부자 동향에만 필요.

## 설계 원칙
- 단일 `index.html`, 서버·빌드 없음 — GitHub Pages에서 그대로 동작
- 모든 사용자 데이터(계좌·일지·API 키)는 브라우저 localStorage에만 저장
- 미국 시장엔 상·하한가가 없음(한국 제도) — 학습 포인트는 지지/저항, 갭, 거래량, 그리고 **근거 있는 매매 습관**

## 학습 철학
데모 계좌의 목적은 수익이 아니라 습관이다:
1. 진입 전에 "왜"를 문장으로 쓸 수 있는가 (일지가 강제함)
2. 청산 후 복기를 남기는가
3. 데모 수익엔 실전 심리(공포·욕심)가 빠져 있음을 아는가
