# CLAUDE.md — Invest Workbench

미국시장 학습용 대시보드 + 데모 트레이딩 + 매매일지. **단일 `index.html`** 하나가 앱 전체다 — 서버·빌드·의존성 없음, GitHub Pages에서 그대로 동작.

## 절대 규칙

- **API 키는 절대 커밋 금지.** Finnhub 키는 사용자가 페이지의 "설정" 섹션에서 입력하고 브라우저 localStorage에만 저장된다. 코드·README·커밋 메시지 어디에도 실제 키를 넣지 말 것. 커밋 전 키 패턴이 섞여 들어갔는지 확인.
- 사용자 데이터(계좌·매매일지·키)는 전부 localStorage(`invest-workbench-v1`)에만 존재한다. 서버로 보내는 코드를 추가하지 말 것.
- 교육용 시뮬레이터라는 정체성 유지 — 투자 권유로 읽힐 문구 금지, 하단 면책 문구 유지.

## 사용자 컨텍스트

- 소유자는 주식 초보 학습자. UI 문구·주석·설명은 쉬운 한국어로, 금융 용어는 풀어서.
- 이 프로젝트의 목적은 수익이 아니라 **매매 습관 훈련**:
  1. 진입 전 "왜"를 문장으로 쓰기 (주문 다이얼로그에서 근거 10자 이상 강제)
  2. 청산 후 복기 남기기 (일지의 복기 칸)
  3. 데모 수익엔 실전 심리(공포·욕심)가 빠져 있음을 인지
- 이 철학을 약화시키는 변경(근거 입력을 선택사항으로 바꾸기 등)은 하지 말 것.
- **학습 사이드노트**: 화면 곳곳의 `ℹ️`/`⚠️` note 박스가 한국(KRX) 제도와 미국 제도의 차이를 짚어준다 (상·하한가 없음, 외국인·기관 일별 순매수 미공개 → SEC Form 4가 프록시 등). 새 섹션을 추가하면 같은 스타일의 학습 노트를 함께 붙이는 것이 이 버전의 디자인 방향.

## 구조 (index.html 내부)

| 영역 | 구현 | 키 필요 |
|---|---|---|
| 티커 스트립·차트·시장개요·급등락·히트맵·경제캘린더 | TradingView 임베드 위젯 (`mountTV()`) | ❌ |
| 시세 조회·데모 매매 평가·내부자 매매(SEC Form 4) | Finnhub REST (`fh()` 헬퍼, 무료 티어 분당 60회) | ✅ |
| 계좌·포지션·매매일지·복기·CSV 내보내기 | 순수 JS + localStorage | ❌ |

- 상태 객체 `S = {cash, positions, trades, key}` — 스키마 바꿀 땐 기존 localStorage 데이터와의 호환(마이그레이션) 고려.
- 디자인: 토스 스타일 다크. CSS 변수(`:root`)로 색 관리 — 상승 빨강(`--up`)·하락 파랑(`--down`)은 한국 관례를 따른 것이니 뒤집지 말 것.
- 한국어 로케일: TradingView `locale:'kr'`, 타임존 `Asia/Seoul`.

## 작업 방식

- 파일이 하나뿐이므로 수정 후 브라우저에서 직접 열어 확인 가능 (빌드 불필요).
- 배포는 GitHub Pages (main / root) — main에 푸시하면 곧 반영됨.
- 미국 제도 관련 학습 노트를 추가·수정할 땐 사실 확인 필수 (예: LULD 서킷브레이커, Form 4 공시 기한 등).
